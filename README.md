# Circle of the Shepherd Helper Tool
This is a small helper tool that I use to manage my summoned creatures when playing a Circle of the Shepherd druid.


## Setup & Run Locally

Setup your virtual environment and install the required libraries.

```
python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
```

Run the app:
```
streamlit run shepherd.py
```
